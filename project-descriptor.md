# displai

A whiteboard-style drawing application with planned features for collaboration and AI interaction.

---

## Version 0.0

**Goal:** Basic drawing functionality in an application window.

### Features
- Application window with a drawable canvas
- Click to draw black pixels
- Continuous drawing as cursor moves while clicked (drag to draw)

### Technical Requirements
- Display a window with a white canvas
- Track mouse click and drag events
- Render black pixels at cursor position during click/drag
